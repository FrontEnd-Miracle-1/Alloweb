# Alloweb

https://alloweb.herokuapp.com/

Kids allowance app - complete family tasks, get allowance money and purchase items from your wishlist.

## Built With

- Angular
- MySQL
- Sequelize
- Cloud Database
- Material Design
- Ebay API
- Multer
